# WARFRAME — PROGRESSION & MAÎTRISE (MASTERY RANK)

This document defines the official Warframe Mastery Rank progression system.
All values are factual and deterministic.

---

## 1. MASTER RANK (MR) — DEFINITION

Mastery Rank (MR) represents overall account progression.
It is increased by gaining Mastery XP from equipment and activities.

Mastery Rank is global and permanent.

---

## 2. MASTER RANK XP SOURCES

Mastery XP is gained ONLY the first time an item is leveled to its maximum rank.

Repeated leveling of the same item does NOT grant additional XP.

---

## 3. XP VALUES PER ITEM TYPE

### WARFRAMES
- Warframe (Rank 30): **6,000 XP**
- Prime Warframe (Rank 30): **6,000 XP**
- Umbra Warframe: **6,000 XP**
- Necramech (Rank 40): **8,000 XP**

### ARCHWINGS
- Archwing (Rank 30): **6,000 XP**

### COMPANIONS
- Companion (Kubrow, Kavat, Sentinel): **6,000 XP**
- Sentinel Weapon: **3,000 XP**
- MOA Companion: **6,000 XP**
- MOA Weapon: **3,000 XP**

### PRIMARY WEAPONS
- Primary Weapon (Rank 30): **3,000 XP**
- Kuva / Tenet Weapon (Rank 40): **4,000 XP**

### SECONDARY WEAPONS
- Secondary Weapon (Rank 30): **3,000 XP**
- Kuva / Tenet Secondary (Rank 40): **4,000 XP**

### MELEE WEAPONS
- Melee Weapon (Rank 30): **3,000 XP**
- Kuva / Tenet Melee (Rank 40): **4,000 XP**

### ARCH-GUNS
- Arch-Gun (Rank 30): **3,000 XP**

---

## 4. SPECIAL SOURCES

### RAILJACK
- Intrinsics (Rank 10 per category): **1,500 XP per rank**
- Total Railjack XP is capped.

### DRIFTER / OPERATOR
- Operator Amps (Rank 30): **3,000 XP**
- Drifter Intrinsics contribute to mastery.

## 6. MASTERY RANK TESTS

- Each Mastery Rank requires a test.
- Tests unlock once sufficient XP is reached.
- Tests can be retried after a cooldown if failed.
- Practice mode is available.

Failing a test does NOT remove XP.

---

## 7. MASTERY RANK BENEFITS

Each Mastery Rank increases:

- Daily standing cap
- Void Trace cap
- Maximum trades per day
- Base mod capacity on unranked items
- Weapon and content access

Higher MR unlocks exclusive weapons and features.

---

## 8. IMPORTANT RULES

- Mastery XP is granted ONCE per item.
- Forma resets item rank but does NOT give additional XP.
- Duplicate items do NOT grant extra XP.
- Prime variants count as separate items.

---

## 9. OPTIMIZATION RULES (PLAYER LOGIC)

Best XP efficiency:
- Level Warframes first (6,000 XP)
- Level companions and sentinels
- Level weapons last
- Kuva / Tenet weapons give extra XP

---

## 10. AI BEHAVIOR RULES

When answering progression questions:
- Always state XP values explicitly
- Never approximate XP thresholds
- Ask user for missing items if calculation is required
- Prefer exact totals over estimates
